# Ganbot Commands

Ganbot is an IRC bot with AI capabilities and image generation features.

## Basic Commands

### Ask Command
Ask the bot a question using AI:
```
!ask <your question>
```

### Prompt Command
Generate images using AI:
```
!prompt <your image description>
```

### Combine Command
Play the word combination game:
```
!combine <word1> <word2>
```
